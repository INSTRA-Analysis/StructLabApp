# StructLab

A 2D structural analysis application for beams, frames, and trusses using the Direct Stiffness Method.

## Tech Stack
- Python 3.12
- NumPy / SciPy — matrix operations
- Matplotlib / Plotly — visualization
- Streamlit — web UI
- pytest — testing

## Run

```bash
# Install dependencies
pip install -r requirements.txt

# Launch the UI
streamlit run ui/app.py

# Run tests
pytest tests/
```
